# Contributing

Terima kasih sudah berkontribusi! Berikut panduannya:

## Proses Kontribusi

1. **Fork** repository ini
2. **Buat branch** fitur: `git checkout -b feat/nama-fitur`
3. **Commit** perubahan: `git commit -m 'feat: menambahkan fitur x'`
4. **Push** ke branch: `git push origin feat/nama-fitur`
5. **Buat Pull Request**

## Aturan Commit

Gunakan [Conventional Commits](https://www.conventionalcommits.org/):

- `feat:` - Fitur baru
- `fix:` - Perbaikan bug
- `docs:` - Dokumentasi
- `refactor:` - Refactoring
- `test:` - Testing
- `chore:` - Tugas maintenance

## Code Style

- Gunakan ESLint + Prettier
- 2 spasi indentasi
- Koma di akhir (trailing comma)
- Single quotes untuk string
