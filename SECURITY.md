# Security Policy

## Reporting a Vulnerability

Jika Anda menemukan kerentanan keamanan, mohon jangan buka issue publik.

Kirim laporan ke email: **security@example.com**

Kami akan merespons dalam 48 jam dan memberikan update setiap 7 hari.

## Apa yang Dilaporkan

- XSS, CSRF, SQL Injection
- Authentication/Authorization bypass
- Data exposure
- Dependency vulnerabilities

## Responsible Disclosure

Mohon berikan waktu 90 hari untuk memperbaiki sebelum mengungkapkan ke publik.
